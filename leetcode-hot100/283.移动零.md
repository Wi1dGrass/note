# [283. 移动零](https://leetcode.cn/problems/move-zeroes/)

给定一个数组 `nums`，将所有 `0` 移动到数组末尾，同时保持非零元素的相对顺序。必须原地操作，不复制数组。

## 思路

1. 用指针 `s` 记录当前非零元素应该放的位置，初始为 0
2. 遍历数组，遇到非零数就放到 `nums[s]` 的位置，然后 `s++`
3. 遍历完成后，从 `s` 到 `n-1` 的位置全部置为 0

时间复杂度：O(n)
空间复杂度：O(1)，原地操作

## 代码

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int s = 0;

        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) {
                nums[s] = nums[i];
                s++;
            }
        }

        for (int i = s; i < nums.length; i++) {
            nums[i] = 0;
        }
    }
}
```
