# [239. 滑动窗口最大值](https://leetcode.cn/problems/sliding-window-maximum/)

给你一个整数数组 `nums`，有一个大小为 `k` 的滑动窗口从最左侧移动到最右侧。滑动窗口每次只向右移动一位，返回滑动窗口中的最大值。

## 笑点分析

这是一个降本增笑的故事：

1. 如果新员工比老员工强（或者一样强），把老员工裁掉。（元素进入窗口）
2. 如果老员工 35 岁了，也裁掉。（元素离开窗口）

裁员后，资历最老（最左边）的人就是最强的员工了。 ——灵茶山艾府

## 思路

使用单调递减队列（双端队列），队列中存的是**下标**，保证队首始终是当前窗口最大值的下标。

1. **右边入（新员工入职）**：当前元素从右侧进入，如果比队列右端的值大（或相等），就把右端的弹出，直到队列为空或右端值更大。保证队列从左到右递减
2. **左边出（老员工退休）**：如果队首下标已经超出窗口左边界（`< left`），说明该元素不在窗口内了，移除队首
3. **记录答案**：当窗口完整形成（`left >= 0`）时，队首就是当前窗口最大值的下标

**易错点**：队列存的是下标而不是值，这样才能判断元素是否还在窗口内

时间复杂度：O(n)，每个元素最多入队出队各一次
空间复杂度：O(k)，队列最多存 k 个元素

## 代码

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        int n = nums.length;
        int[] ans = new int[n - k + 1]; // 窗口共滑动 n - k + 1 次
        Deque<Integer> q = new ArrayDeque<>(); // 单调递减队列，存下标

        for (int i = 0; i < n; i++) {
            // 1. 右边入：把比当前元素小的都裁掉，保持单调递减
            while (!q.isEmpty() && nums[q.getLast()] <= nums[i]) {
                q.removeLast();
            }
            q.addLast(i); // 当前元素入队

            // 2. 左边出：队首下标超出窗口范围，移除
            int left = i - k + 1; // 窗口左边界
            if (q.getFirst() < left) {
                q.removeFirst();
            }

            // 3. 窗口完整时，队首就是最大值的下标
            if (left >= 0) {
                ans[left] = nums[q.getFirst()];
            }
        }

        return ans;
    }
}
```
