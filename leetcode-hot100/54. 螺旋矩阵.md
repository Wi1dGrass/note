# [54. 螺旋矩阵](https://leetcode.cn/problems/spiral-matrix/)

给你一个 `m` 行 `n` 列的矩阵 `matrix`，按照顺时针螺旋顺序返回矩阵中的所有元素。

## 思路

### 方法一：按层模拟

用四个变量 `up`、`down`、`left`、`right` 表示当前遍历的边界，每完成一个方向就收缩对应的边界，顺时针遍历顺序：左→右、上→下、右→左、下→上。

**易错点**：从右到左和从下到上之前要检查边界是否交叉（`up <= down` 和 `left <= right`），否则会重复添加元素。

时间复杂度：O(m × n)
空间复杂度：O(1)，不考虑返回值

```java
class Solution {
    public List<Integer> spiralOrder(int[][] matrix) {
        int m = matrix.length, n = matrix[0].length;
        int up = 0, down = m - 1, left = 0, right = n - 1;
        List<Integer> list = new ArrayList<>();

        while (up <= down && left <= right) {
            // 从左到右（上边界）
            for (int i = left; i <= right; i++) {
                list.add(matrix[up][i]);
            }
            up++; // 上边界下移

            // 从上到下（右边界）
            for (int i = up; i <= down; i++) {
                list.add(matrix[i][right]);
            }
            right--; // 右边界左移

            // 从右到左（下边界），需检查上下边界是否交叉
            if (up <= down) {
                for (int i = right; i >= left; i--) {
                    list.add(matrix[down][i]);
                }
                down--; // 下边界上移
            }

            // 从下到上（左边界），需检查左右边界是否交叉
            if (left <= right) {
                for (int i = down; i >= up; i--) {
                    list.add(matrix[i][left]);
                }
                left++; // 左边界右移
            }
        }

        return list;
    }
}
```

### 方法二：方向数组

定义方向数组 `{{0,1},{1,0},{0,-1},{-1,0}}` 表示右、下、左、上，走到边界或已访问的位置时右转 90°（`di = (di + 1) % 4`）。用 `Integer.MAX_VALUE` 标记已访问的元素。

时间复杂度：O(m × n)
空间复杂度：O(1)

```java
class Solution {
    private static final int[][] DIRS = {{0, 1}, {1, 0}, {0, -1}, {-1, 0}}; // 右、下、左、上

    public List<Integer> spiralOrder(int[][] matrix) {
        int m = matrix.length, n = matrix[0].length;
        List<Integer> ans = new ArrayList<>(m * n);
        int i = 0, j = 0, di = 0;

        for (int k = 0; k < m * n; k++) {
            ans.add(matrix[i][j]);
            matrix[i][j] = Integer.MAX_VALUE; // 标记已访问

            // 检查下一步是否越界或已访问
            int x = i + DIRS[di][0];
            int y = j + DIRS[di][1];
            if (x < 0 || x >= m || y < 0 || y >= n || matrix[x][y] == Integer.MAX_VALUE) {
                di = (di + 1) % 4; // 右转 90°
            }

            i += DIRS[di][0]; // 按新方向走一步
            j += DIRS[di][1];
        }

        return ans;
    }
}
```
