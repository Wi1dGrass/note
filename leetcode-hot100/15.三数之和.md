# [15. 三数之和](https://leetcode.cn/problems/3sum/)

给你一个整数数组 `nums`，返回所有和为 0 且不重复的三元组。

## 思路

1. **排序**：先对数组排序，使其有序，方便去重和使用双指针（为什么能排序，因为答案对数组的顺序不感冒的）
2. **转换为两数之和**：固定一个数 `nums[i]`，问题转化为在剩余部分找两个数之和等于 `-nums[i]`
3. **双指针查找**：左指针指向 `i + 1`（较小值），右指针指向 `n - 1`（较大值）
   - `sum > target`：和太大，右指针左移
   - `sum < target`：和太小，左指针右移
   - `sum == target`：记录答案，左右指针同时收缩
4. **去重**：三处需要去重
   - 固定 `nums[i]` 时：跳过与前一个相同的值，避免重复三元组
   - 找到答案后：左指针跳过连续相同的值
   - 找到答案后：右指针跳过连续相同的值

时间复杂度：O(n²)，排序 O(n log n) + 外层循环 × 内层双指针 O(n²)
空间复杂度：O(1)，不考虑返回值

## 代码

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> ans = new ArrayList<>();
        int n = nums.length;
        Arrays.sort(nums);

        for (int i = 0; i < n - 2; i++) {
            if (i > 0 && nums[i] == nums[i - 1]) continue; // 固定数去重

            int target = -nums[i];
            int left = i + 1, right = n - 1;

            while (left < right) {
                int sum = nums[left] + nums[right];
                if (sum == target) {
                    ans.add(List.of(nums[i], nums[left], nums[right]));
                    left++;
                    right--;
                    while (left < right && nums[left] == nums[left - 1]) left++;   // 左指针去重
                    while (left < right && nums[right] == nums[right + 1]) right--; // 右指针去重
                } else if (sum > target) {
                    right--;
                } else {
                    left++;
                }
            }
        }

        return ans;
    }
}
```
