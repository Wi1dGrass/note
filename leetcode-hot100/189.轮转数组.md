# [189. 轮转数组](https://leetcode.cn/problems/rotate-array/)

给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置。

## 思路

### 方法一：临时数组

1. `k %= n`：轮转 `k` 次等于轮转 `k % n` 次
2. 用临时数组存储后 `k` 个元素（会被覆盖的元素）
3. 将前 `n - k` 个元素右移
4. 将临时数组中的元素放回数组开头

时间复杂度：O(n)
空间复杂度：O(n)，临时数组

```java
class Solution {
    public void rotate(int[] nums, int k) {
        int n = nums.length;
        k %= n; // 轮转 k 次等于轮转 k % n 次

        int[] temp = new int[k];
        // 存储最后 k 个元素
        for (int i = 0; i < k; i++) {
            temp[i] = nums[n - k + i];
        }

        // 前 n - k 个元素右移
        for (int i = n - 1; i >= k; i--) {
            nums[i] = nums[i - k];
        }

        // 临时数组元素放回开头
        for (int i = 0; i < k; i++) {
            nums[i] = temp[i];
        }
    }
}
```

### 方法二：数组翻转

1. 整体翻转：`[1,2,3,4,5,6,7]` → `[7,6,5,4,3,2,1]`
2. 翻转前 k 个：`[7,6,5,4,3,2,1]` → `[5,6,7,4,3,2,1]`
3. 翻转后 n-k 个：`[5,6,7,4,3,2,1]` → `[5,6,7,1,2,3,4]`

为什么这样有效？整体翻转后，前 k 个和后 n-k 个元素的位置是对的，但各自内部是反的，再分别翻转即可。

时间复杂度：O(n)
空间复杂度：O(1)

```java
class Solution {
    public void rotate(int[] nums, int k) {
        int n = nums.length;
        k %= n;

        reverse(nums, 0, n - 1);  // 整体翻转
        reverse(nums, 0, k - 1);  // 翻转前 k 个
        reverse(nums, k, n - 1);  // 翻转后 n-k 个
    }

    private void reverse(int[] nums, int i, int j) {
        while (i < j) {
            int temp = nums[i];
            nums[i++] = nums[j];
            nums[j--] = temp;
        }
    }
}
```
