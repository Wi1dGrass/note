# [160. 相交链表](https://leetcode.cn/problems/intersection-of-two-linked-lists/)

给你两个单链表的头节点 `headA` 和 `headB`，找出并返回两个单链表相交的起始节点。如果不存在相交节点，返回 `null`。

![lc160-3-c.png](https://pic.leetcode.cn/1729473968-TLOxoH-lc160-3-c.png)

## 思路

设链表 A 长度为 x + z，链表 B 长度为 y + z，其中 z 为公共部分。

1. **x == y**：两个指针同时到达相交点
2. **x != y**：短链表的指针先跑完，让它从另一条链表头部继续跑。最终两个指针都走了 x + y + z 步，在相交点相遇

走完自己的链表后切换到另一条链表，相当于**消除了长度差**，两个指针最终会在相交点相遇（或同时到达 null）。

时间复杂度：O(m + n)
空间复杂度：O(1)

## 代码

```java
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        ListNode p1 = headA, p2 = headB;

        while (p1 != p2) {
            // 走完当前链表就切换到另一条链表，消除长度差
            p1 = p1 != null ? p1.next : headB;
            p2 = p2 != null ? p2.next : headA;
        }

        return p1; // 相交节点或 null
    }
}
```
