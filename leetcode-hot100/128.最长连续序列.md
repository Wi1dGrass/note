# [128. 最长连续序列](https://leetcode.cn/problems/longest-consecutive-sequence/)

给定一个未排序的整数数组 `nums`，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

要求时间复杂度 O(n)。

## 思路

1. 使用 HashSet 对数组去重，同时实现 O(1) 查找
2. 遍历 set 中的每个数 n，判断它能否作为一个连续序列的**起始数**：即 set 中不存在 `n - 1`（如 n=100，若 99 存在则 100 不是起始数，跳过）
3. 确定起始数后，不断查找 `currentNum + 1` 是否在 set 中，每找到一个长度 +1，直到找不到为止
4. 记录所有序列中的最大长度

为什么要判断起始数？避免重复计算。比如 [1,2,3,4]，如果从 1 开始已经算出了长度 4，后续遍历到 2、3、4 时就不需要再算了。

时间复杂度：O(n)，虽然有两层循环，但每个数最多被内层 while 访问一次
空间复杂度：O(n)，HashSet 存储

## 代码

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        if (nums.length == 0) {
            return 0;
        }

        Set<Integer> set = new HashSet<>();
        for (int n : nums) {
            set.add(n);
        }

        int maxLen = 0;

        for (int n : set) {
            // 只有当 n 是序列起始数时才开始计算
            if (!set.contains(n - 1)) {
                int currentNum = n;
                int len = 1;

                while (set.contains(currentNum + 1)) {
                    len++;
                    currentNum++;
                }

                maxLen = Math.max(maxLen, len);
            }
        }

        return maxLen;
    }
}
```
