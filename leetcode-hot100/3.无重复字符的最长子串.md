# [3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

给定一个字符串 `s`，找出其中不含有重复字符的最长子串的长度。

## 思路

1. **如何判断重复**：用长度为 128 的 int 数组记录每个 ASCII 字符出现的次数（也可以用 HashMap，数组更快）
2. **滑动窗口**：定义左右指针 `i` 和 `j`，维护窗口 `[i, j]` 内无重复字符
   - **扩张右边界**：如果当前字符不重复（出现次数 <= 1），直接纳入窗口
   - **收缩左边界**：如果当前字符重复了（出现次数 > 1），不断移动左指针并减少左边字符的计数，直到窗口内无重复
3. 每次扩张后更新最长长度 `ans = max(ans, j - i + 1)`

时间复杂度：O(n)，左右指针各最多遍历一次
空间复杂度：O(1)，固定大小的 ASCII 数组

## 代码

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        int ans = 0;
        int[] ascii = new int[128]; // 记录每个字符出现的次数
        char[] strs = s.toCharArray();
        int i = 0, j = 0; // i = 左边界，j = 右边界

        while (j < strs.length) {
            ascii[strs[j]]++; // 右边界字符加入窗口

            // 当前字符重复了，收缩左边界直到无重复
            while (ascii[strs[j]] > 1) {
                ascii[strs[i]]--; // 左边界字符移出窗口
                i++;              // 左边界右移
            }

            // 更新最长无重复子串长度（j 还没++，所以要 +1）
            ans = Math.max(ans, j - i + 1);
            j++; // 右边界右移，继续扩张
        }

        return ans;
    }
}
```
