# [240. 搜索二维矩阵 II](https://leetcode.cn/problems/search-a-2d-matrix-ii/)

编写一个高效的算法来搜索 m × n 矩阵中的一个目标值 `target`。矩阵特性：每行从左到右升序，每列从上到下升序。

## 思路

从右上角开始，利用矩阵的有序性排除：

1. 当前值 < target：当前行左边的值更小，整行排除，`i++` 下移
2. 当前值 > target：当前列下边的值更大，整列排除，`j--` 左移
3. 当前值 == target：找到答案

为什么从右上角开始？因为右上角是一行中最大的、一列中最小的，每次比较都能排除一行或一列。从左上角则无法排除（右边和下边都比它大）。

时间复杂度：O(m + n)，最多走 m + n 步
空间复杂度：O(1)

## 代码

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int i = 0;                            // 从第一行开始
        int j = matrix[0].length - 1;         // 从最后一列开始（右上角）

        while (i < matrix.length && j >= 0) {
            if (matrix[i][j] == target) {
                return true;
            }
            if (matrix[i][j] < target) {
                i++;  // 当前值太小，排除这一行
            } else {
                j--;  // 当前值太大，排除这一列
            }
        }

        return false;
    }
}
```

## 扩展：[1351. 统计有序矩阵中的负数](https://leetcode.cn/problems/count-negative-numbers-in-a-sorted-matrix/)

给你一个 m × n 的矩阵 grid，元素按行按列非严格递减，统计负数的数目。

### 思路

同样是利用矩阵有序性。从右上角开始，遇到负数说明该列往下全是负数，直接统计后左移；遇到非负数说明该行还没到负数区域，下移继续找。

时间复杂度：O(m + n)
空间复杂度：O(1)

### 代码

```java
class Solution {
    public int countNegatives(int[][] grid) {
        int i = 0;
        int j = grid[0].length - 1; // 从右上角开始
        int ans = 0;

        while (i < grid.length && j >= 0) {
            if (grid[i][j] < 0) {
                // 当前位置是负数，该列从 i 到末尾全是负数
                ans += grid.length - i;
                j--; // 左移到上一列
            } else {
                i++; // 还没到负数区域，下移
            }
        }

        return ans;
    }
}
```
