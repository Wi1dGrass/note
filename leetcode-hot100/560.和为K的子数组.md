# [560. 和为 K 的子数组](https://leetcode.cn/problems/subarray-sum-equals-k/)

给你一个整数数组 `nums` 和一个整数 `k`，统计并返回该数组中和为 `k` 的子数组的个数。子数组是数组中元素的连续非空序列。

## 思路

### 暴力枚举

两层循环遍历所有子数组，判断和是否等于 k。时间 O(n²)，空间 O(1)。

### 前缀和 + 哈希表

1. **前缀和公式**：`prefix[j] - prefix[i] = k`，说明区间 `[i+1, j]` 的子数组和为 k
2. **变形公式**：`prefix[i] = prefix[j] - k`，即遍历到 j 时，只需要找之前有多少个前缀和等于 `prefix[j] - k`
3. **哈希表优化**：用 HashMap 记录每个前缀和出现的次数，遍历时查 `map.get(sum - k)` 就能得到以当前结尾的合法子数组个数
4. **初始化 `map.put(0, 1)`**：表示前缀和为 0 出现了 1 次，这样当 `sum == k` 即 `sum - k == 0` 时能正确计数

**易错点**：必须先查询再更新 map，否则会把当前元素也算进去

时间复杂度：O(n)，一次遍历
空间复杂度：O(n)，HashMap 存储

## 代码

```java
class Solution {
    public int subarraySum(int[] nums, int k) {
        // key = 前缀和，value = 该前缀和出现的次数
        HashMap<Integer, Integer> map = new HashMap<>();
        map.put(0, 1); // 前缀和为 0 出现 1 次，处理 sum == k 的情况

        int ans = 0, sum = 0;

        for (int num : nums) {
            sum += num; // 计算当前前缀和

            // 查找之前有多少个前缀和等于 sum - k
            // 每找到一个，就对应一个以当前元素结尾的合法子数组
            ans += map.getOrDefault(sum - k, 0);

            // 将当前前缀和加入 map（必须在查询之后）
            map.merge(sum, 1, Integer::sum);
        }

        return ans;
    }
}
```
