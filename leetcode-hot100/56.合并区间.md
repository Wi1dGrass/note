# [56. 合并区间](https://leetcode.cn/problems/merge-intervals/)

以数组 `intervals` 表示若干个区间的集合，合并所有重叠的区间，返回一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间。

## 思路

1. **排序**：按区间左端点升序排序，保证区间有序
2. **合并判断**：
   - 重叠条件：当前区间左端点 <= 已合并区间的右端点
   - 合并操作：新区间的左端点不变，右端点 = `max(当前右端点, 新区间右端点)`
   - 不重叠：保存当前合并区间，重置为新区间
3. **注意边界**：循环结束后最后一个区间需要手动加入答案

时间复杂度：O(n log n)，排序占主导
空间复杂度：O(log n)，排序的递归栈（不考虑返回值）

## 代码

```java
class Solution {
    public int[][] merge(int[][] intervals) {
        if (intervals == null || intervals.length == 0) {
            return new int[0][0];
        }

        // 按左端点升序排序
        Arrays.sort(intervals, (a, b) -> Integer.compare(a[0], b[0]));

        List<int[]> merged = new ArrayList<>();
        int start = intervals[0][0];
        int end = intervals[0][1];

        for (int i = 1; i < intervals.length; i++) {
            if (intervals[i][0] <= end) {
                // 重叠，合并：更新右端点为较大值
                end = Math.max(end, intervals[i][1]);
            } else {
                // 不重叠，保存当前区间并重置
                merged.add(new int[]{start, end});
                start = intervals[i][0];
                end = intervals[i][1];
            }
        }

        // 最后一个区间加入结果
        merged.add(new int[]{start, end});

        return merged.toArray(new int[merged.size()][]);
    }
}
```
