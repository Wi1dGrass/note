# [42. 接雨水](https://leetcode.cn/problems/trapping-rain-water/)

给定 `n` 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

## 思路

核心原理：位置 i 能接的雨水量 = `min(左边最高柱子, 右边最高柱子) - height[i]`，即木桶效应，能接多少水取决于较矮的那一边。

---

### 方法一：动态规划

1. 分别从左到右和从右到左遍历，预处理出两个数组：
   - `left_max[i]`：位置 i 左边（含自身）的最高柱子
   - `right_max[i]`：位置 i 右边（含自身）的最高柱子
2. 再遍历一次，每个位置接水量 = `min(left_max[i], right_max[i]) - height[i]`

时间复杂度：O(n)，三次遍历
空间复杂度：O(n)，两个辅助数组

```java
class Solution {
    public int trap(int[] height) {
        int n = height.length;
        int[] leftMax = new int[n];
        int[] rightMax = new int[n];

        // 从左到右，记录左边最大高度
        int max = 0;
        for (int i = 0; i < n; i++) {
            max = Math.max(max, height[i]);
            leftMax[i] = max;
        }

        // 从右到左，记录右边最大高度
        max = 0;
        for (int i = n - 1; i >= 0; i--) {
            max = Math.max(max, height[i]);
            rightMax[i] = max;
        }

        // 计算每个位置能接的雨水量，木桶效应取较小值
        int area = 0;
        for (int i = 0; i < n; i++) {
            area += Math.min(leftMax[i], rightMax[i]) - height[i];
        }

        return area;
    }
}
```

### 方法二：双指针（动态规划优化）

1. 动态规划用了两个 O(n) 数组，但其实不需要同时知道两侧完整的最大值，只需要知道**较小的一侧**的最大值就能确定接水量
2. 定义左右指针和 `leftMax`、`rightMax`，每次比较两侧最大值，**维护较小的一侧**向内移动
3. 为什么可以这样做？假设 `leftMax < rightMax`，那么位置 left 的接水量只取决于 `leftMax`（因为右边一定有一个比它高的柱子），所以可以直接计算

**易错点**：先移动指针再更新最大值和计算接水量的顺序不能乱

时间复杂度：O(n)
空间复杂度：O(1)，双指针省去了两个数组

```java
class Solution {
    public int trap(int[] height) {
        int left = 0, right = height.length - 1;
        int ans = 0;
        int leftMax = height[left], rightMax = height[right];

        while (left < right) {
            if (leftMax < rightMax) {
                left++;
                leftMax = Math.max(leftMax, height[left]);
                ans += leftMax - height[left];
            } else {
                right--;
                rightMax = Math.max(rightMax, height[right]);
                ans += rightMax - height[right];
            }
        }

        return ans;
    }
}
```

### 方法三：单调栈

1. 维护一个从栈底到栈顶**递减**的单调栈（存的是下标）
2. 当当前柱子高度 `h` 大于栈顶时，说明栈顶柱子可能形成凹槽的底部
3. 弹出栈顶作为底部 `bottomH`，新的栈顶就是左边界 `left`，当前柱子是右边界 `i`
4. 计算这一层接水量：高 = `min(height[left], height[i]) - bottomH`，宽 = `i - left - 1`

**易错点**：弹出栈顶后要检查栈是否为空，空栈说明没有左边界，无法接水

时间复杂度：O(n)，每个元素最多入栈出栈各一次
空间复杂度：O(n)

```java
class Solution {
    public int trap(int[] height) {
        int ans = 0;
        Deque<Integer> stack = new ArrayDeque<>();

        for (int i = 0; i < height.length; i++) {
            int h = height[i];
            while (!stack.isEmpty() && height[stack.peek()] <= h) {
                int bottomH = height[stack.pop()];
                if (stack.isEmpty()) break; // 没有左边界，无法接水
                int left = stack.peek();
                int dh = Math.min(height[left], h) - bottomH;
                ans += dh * (i - left - 1);
            }
            stack.push(i);
        }

        return ans;
    }
}
```
