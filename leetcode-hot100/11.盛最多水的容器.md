# [11. 盛最多水的容器](https://leetcode.cn/problems/container-with-most-water/)

给定一个长度为 `n` 的整数数组 `height`，找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。返回容器可以储存的最大水量。

## 思路

1. 容积 = 高 × 宽，高由两根柱子中较矮的决定（`min(h[left], h[right])`），宽为两根柱子的距离（`right - left`）
2. 定义左右指针，左指针指向数组第一个元素，右指针指向最后一个元素，此时宽度最大
3. 每次计算当前容积后，**移动较矮的那根柱子**向内收缩：
   - 如果移动较高的柱子，宽度变小而高不会超过较矮的柱子，容积一定变小，所以没有意义
   - 移动较矮的柱子才有可能找到更高的柱子，从而获得更大的容积
4. 记录过程中的最大值

时间复杂度：O(n)，双指针各遍历一次
空间复杂度：O(1)

## 代码

```java
class Solution {
    public int maxArea(int[] height) {
        int left = 0, right = height.length - 1;
        int ans = 0;

        while (left < right) {
            int area = (right - left) * Math.min(height[left], height[right]);
            ans = Math.max(ans, area);

            if (height[left] < height[right]) {
                left++;
            } else {
                right--;
            }
        }

        return ans;
    }
}
```
