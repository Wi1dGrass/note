# [438. 找到字符串中所有字母异位词](https://leetcode.cn/problems/find-all-anagrams-in-a-string/)

给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的异位词子串，返回这些子串的起始索引。

## 思路

1. **记录目标频次**：用 HashMap 记录 `p` 中每个字符出现的次数
2. **滑动窗口**：窗口大小固定为 `p.length()`，维护窗口 `[i, j]`
   - 字符进入窗口（右边界 `j` 右移）：对应字符计数 -1
   - 字符离开窗口（左边界 `i` 右移）：对应字符计数 +1
3. **判断匹配**：当窗口大小等于 `p.length()` 时，检查 HashMap 中所有 value 是否为 0，是则说明窗口内字符频次与 `p` 完全一致，记录起始索引

**易错点**：字符离开窗口时要恢复计数（+1），否则后续判断会出错

时间复杂度：O(n × m)，n 为 s 长度，m 为 p 中不同字符数（每次匹配检查）
空间复杂度：O(m)，HashMap 存储 p 的字符

## 代码

```java
class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        HashMap<Character, Integer> map = new HashMap<>();
        List<Integer> ans = new ArrayList<>();
        char[] strS = s.toCharArray();
        char[] strP = p.toCharArray();

        // 记录 p 中每个字符的出现频次（目标频次）
        for (char c : strP) {
            map.put(c, map.getOrDefault(c, 0) + 1);
        }

        int i = 0, j = 0; // i = 左边界，j = 右边界

        while (j < strS.length) {
            // 右边界字符进入窗口，对应计数 -1（表示匹配掉了一个）
            if (map.containsKey(strS[j])) {
                map.put(strS[j], map.get(strS[j]) - 1);
            }

            // 窗口大小等于 p 的长度时，判断是否匹配
            if (j - i + 1 == p.length()) {
                // 检查所有计数是否为 0（窗口内字符频次与 p 完全一致）
                boolean match = true;
                for (int v : map.values()) {
                    if (v != 0) { match = false; break; }
                }
                if (match) ans.add(i); // 匹配成功，记录起始索引

                // 左边界字符离开窗口，恢复计数 +1
                if (map.containsKey(strS[i])) {
                    map.put(strS[i], map.get(strS[i]) + 1);
                }
                i++; // 左边界右移，保持窗口大小
            }

            j++; // 右边界右移，继续扩张
        }

        return ans;
    }
}
```
