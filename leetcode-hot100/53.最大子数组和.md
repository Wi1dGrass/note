# [53. 最大子数组和](https://leetcode.cn/problems/maximum-subarray/)

给你一个整数数组 `nums`，找出一个具有最大和的连续子数组，返回其最大和。

## 思路一：前缀和

1. 最大子数组和 = 前缀和 - 之前最小的前缀和（在当前位置之前找最小的前缀和，相减就是以当前位置结尾的最大子数组和）
2. 维护两个变量：`preSum`（当前前缀和）和 `minPreSum`（历史最小前缀和）
3. `minPreSum` 初始化为 0，表示空前缀（子数组从第一个元素开始的情况）

时间复杂度：O(n)
空间复杂度：O(1)

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int ans = Integer.MIN_VALUE;
        int preSum = 0;     // 当前前缀和
        int minPreSum = 0;  // 历史最小前缀和，初始化为 0（空前缀）

        for (int num : nums) {
            preSum += num;                              // 计算当前前缀和
            ans = Math.max(ans, preSum - minPreSum);    // 最大子数组和 = 当前前缀和 - 最小前缀和
            minPreSum = Math.min(minPreSum, preSum);    // 更新最小前缀和
        }

        return ans;
    }
}
```

## 思路二：动态规划

1. 定义 `f[i]` 表示以 `nums[i]` 结尾的最大子数组和
2. 状态转移：`f[i] = nums[i] + max(f[i-1], 0)`，即要么接上前面的子数组，要么从当前重新开始
3. 答案是所有 `f[i]` 中的最大值		

```
class Solution {
    public int maxSubArray(int[] nums) {
        int[] f = new int[nums.length];
        f[0] = nums[0];
        int ans = f[0];
        for (int i = 1; i < nums.length;i++) {
            f[i] = Math.max(f[i-1],0) + nums[i];
            ans = Math.max(ans, f[i]);
        }
        return ans;
    }
}
```

**优化**：`f[i]` 只依赖 `f[i-1]`，用一个变量 `f` 即可，省去数组

时间复杂度：O(n)
空间复杂度：O(1)

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int ans = Integer.MIN_VALUE; // 注意答案可以是负数，不能初始化为 0
        int f = 0; // f = 以当前元素结尾的最大子数组和

        for (int x : nums) {
            f = Math.max(f, 0) + x; // 接上前面的 or 从当前重新开始
            ans = Math.max(ans, f); // 更新全局最大值
        }

        return ans;
    }
}
```
