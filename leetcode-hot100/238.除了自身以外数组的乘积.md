# [238. 除自身以外数组的乘积](https://leetcode.cn/problems/product-of-array-except-self/)

给定一个整数数组 `nums`，返回数组 `answer`，其中 `answer[i]` 等于 `nums` 中除 `nums[i]` 之外其余各元素的乘积。不要使用除法，在 O(n) 内完成。

## 思路

1. **前缀积 `pre[i]`**：`nums[0]` 到 `nums[i-1]` 的乘积，即 i 左边所有元素的积
2. **后缀积 `suf[i]`**：`nums[i+1]` 到 `nums[n-1]` 的乘积，即 i 右边所有元素的积
3. 答案：`answer[i] = pre[i] * suf[i]`，左边乘右边就是除自身以外的乘积

边界条件：`pre[0] = 1`（左边没有元素），`suf[n-1] = 1`（右边没有元素）

时间复杂度：O(n)，三次遍历
空间复杂度：O(n)，两个前缀积数组

## 代码

```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int n = nums.length;

        // 前缀积：pre[i] = nums[0] * ... * nums[i-1]
        int[] pre = new int[n];
        pre[0] = 1;
        for (int i = 1; i < n; i++) {
            pre[i] = pre[i - 1] * nums[i - 1];
        }

        // 后缀积：suf[i] = nums[i+1] * ... * nums[n-1]
        int[] suf = new int[n];
        suf[n - 1] = 1;
        for (int i = n - 2; i >= 0; i--) {
            suf[i] = suf[i + 1] * nums[i + 1];
        }

        // 答案：answer[i] = pre[i] * suf[i]
        int[] ans = new int[n];
        for (int i = 0; i < n; i++) {
            ans[i] = pre[i] * suf[i];
        }

        return ans;
    }
}
```
