# [141. 环形链表](https://leetcode.cn/problems/linked-list-cycle/)

给你一个链表的头节点 `head`，判断链表中是否有环。

## 思路

快慢指针，快指针走两步，慢指针走一步：

- **有环**：快指针一定能追上慢指针（快指针每轮比慢指针多走一步，距离逐步缩短，必然相遇）
- **无环**：快指针会走到 `null`

时间复杂度：O(n)
空间复杂度：O(1)

## 代码

```java
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) {
 *         val = x;
 *         next = null;
 *     }
 * }
 */
public class Solution {
    public boolean hasCycle(ListNode head) {
        ListNode fast = head;
        ListNode slow = head;

        while (fast != null && fast.next != null) {
            slow = slow.next;          // 慢指针走一步
            fast = fast.next.next;     // 快指针走两步
            if (fast == slow) {
                return true;           // 相遇，说明有环
            }
        }

        return false; // 快指针走到 null，无环
    }
}
```
