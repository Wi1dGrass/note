# [206. 反转链表](https://leetcode.cn/problems/reverse-linked-list/)

给你单链表的头节点 `head`，反转链表并返回反转后的链表。

## 思路

![img](https://assets.leetcode.com/uploads/2021/02/19/rev1ex1.jpg)

核心操作：把当前节点的 next 指向前一个节点。用三个指针 `pre`、`cur`、`tmp` 配合完成：

1. `tmp` 保存 `cur.next`（防止断链）
2. `cur.next = pre`（反转指向）
3. `pre` 和 `cur` 同时后移一步

时间复杂度：O(n)
空间复杂度：O(1)

## 代码

```java
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode pre = null;  // 前一个节点，初始为 null
        ListNode cur = head;  // 当前节点

        while (cur != null) {
            ListNode tmp = cur.next; // 暂存下一个节点，防止断链
            cur.next = pre;          // 反转：当前节点指向前一个
            pre = cur;               // pre 后移
            cur = tmp;               // cur 后移
        }

        return pre; // pre 指向反转后的头节点
    }
}
```
