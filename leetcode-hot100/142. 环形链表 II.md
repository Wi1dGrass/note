# [142. 环形链表 II](https://leetcode.cn/problems/linked-list-cycle-ii/)

给定一个链表的头节点 `head`，返回链表开始入环的第一个节点。如果链表无环，则返回 `null`。不允许修改链表。

## 思路

### 方法一：HashSet

遍历链表，用 HashSet 记录访问过的节点，第一次遇到已访问的节点就是入环节点。

时间复杂度：O(n)
空间复杂度：O(n)

```java
public class Solution {
    public ListNode detectCycle(ListNode head) {
        ListNode p = head;
        HashSet<ListNode> visited = new HashSet<>();

        while (p != null) {
            if (visited.contains(p)) {
                return p; // 第一次重复访问的节点就是入环节点
            }
            visited.add(p);
            p = p.next;
        }

        return null;
    }
}
```

### 方法二：快慢指针

设入环需要走 a 步，环长为 c，相遇时慢指针走了 b 步。

推导：快指针比慢指针多走了 k 圈，`2b = kc + b` => `b = kc`

相遇点距离入环点：慢指针在环中走了 `b - a = kc - a` 步，说明再走 a 步即可到达入环点。

所以相遇后，让一个指针从 head 出发，另一个从相遇点出发，两者同时走，再次相遇就是入环节点。

时间复杂度：O(n)
空间复杂度：O(1)

```java
public class Solution {
    public ListNode detectCycle(ListNode head) {
        ListNode fast = head;
        ListNode slow = head;

        while (fast != null && fast.next != null) {
            slow = slow.next;          // 慢指针走一步
            fast = fast.next.next;     // 快指针走两步
            if (fast == slow) {
                // 相遇后，一个从 head 出发，一个从相遇点出发，再次相遇就是入环节点
                while (head != slow) {
                    head = head.next;
                    slow = slow.next;
                }
                return slow;
            }
        }

        return null; // 无环
    }
}
```
