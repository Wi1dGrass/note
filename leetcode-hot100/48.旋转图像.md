# [48. 旋转图像](https://leetcode.cn/problems/rotate-image/)

给定一个 n × n 的二维矩阵表示一个图像，请顺时针旋转 90 度。必须在原地旋转，不要使用另一个矩阵。

## 思路

顺时针旋转 90° = **转置** + **水平镜像**

1. **转置**：沿主对角线翻转，交换 `matrix[i][j]` 和 `matrix[j][i]`（注意 j < i，只遍历下三角）
2. **水平镜像**：每行左右翻转，交换 `matrix[i][j]` 和 `matrix[i][n-1-j]`（注意 j < n/2）

扩展记忆：
- 顺时针 90°：转置 + 水平镜像
- 逆时针 90°：转置 + 垂直镜像
- 180°：水平镜像 + 垂直镜像

时间复杂度：O(n²)
空间复杂度：O(1)

## 代码

```java
class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;

        // 1. 转置：沿主对角线翻转，j < i 只遍历下三角
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < i; j++) {
                int tmp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = tmp;
            }
        }

        // 2. 水平镜像：每行左右翻转，j < n/2 只遍历左半部分
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n / 2; j++) {
                int tmp = matrix[i][j];
                matrix[i][j] = matrix[i][n - 1 - j];
                matrix[i][n - 1 - j] = tmp;
            }
        }
    }
}
```
